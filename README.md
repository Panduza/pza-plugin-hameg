# pza-plugin-hameg




## HM7044

On old Power Supply there is only a serial port (no usb IDs available)

```json
{
    "devices": [
        {
            "name": "power",
            "dref": "hameg.HM7044",
            "settings": {
                "serial_port_name": "COM9"
            }
        }
    ]
}
```
